# Домашнее задание #10

**Что сделать:** Параллельное копирование больших блоков данных при помощи пула потоков

**Шаги:**
1. Завести пул из заданного числа потоков
2. Реализовать функцию
```c
void* parallel_memcpy(void* dst, const void* src,
size_t size)
```
3. При копировании потоки пула помогают потоку
приложения
4. Для простоты:
  - приложение однопоточное
  - dst и src не перекрываются
5. Протестировать корректность
  - Совпадение исходных и скопированных данных
6. Измерить и сравнить производительность
  - Размер копируемых данных 256М
  - В пуле 0...8 потоков
