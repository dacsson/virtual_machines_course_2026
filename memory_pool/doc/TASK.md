# Домашнее задание No6
● Реализация растущего вниз последовательного

пула для элементов списка
● Тестовая программа строит односвязный список
заданной длины, потом его уничтожает, измеряет
расход времени и памяти
• Программа лежит в поддиректории memory_pool
https://github.com/OlegPliss/virtual_machines
● Измерить расход времени и памяти для списка
длиной 10М
● Реализовать последовательный растущий вниз пул с
защитой памяти для проверки переполнения
● Адресное пространство отводить функцией mmap
● Заменить аллокатор элементов списка на этот пул
● Вместо поэлементного освобождения списка
освобождать весь пул

● Повторить измерения, сравнить результаты
● Рекомендуемые для реализации языки: C, C+
● Рекомендуемая ОС: любой настоящий POSIX
